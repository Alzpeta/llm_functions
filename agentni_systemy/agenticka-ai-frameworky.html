OpenClaw

Rakouský „vibe coder" Peter Steinberger – lokální, stavový agent s důrazem na transparentnost a proaktivitu.

Listopad 2025
Technologie
Open source (TypeScript / Node.js), lokální a stavový běh.
Autonomie
Nativně zmocněný k akci a externí komunikaci jménem uživatele. Umí ukládat přihlašovací údaje.
Paměť
Pamatuje si konverzace v Markdown a YAML – lidsky čitelná a auditovatelná, ne schovaná ve vektorové DB.
Proaktivita
Proaktivní, nejen reaktivní – agent periodicky čte HEARTBEAT.md.
Nástroje
Instrukce a metadata pro použití nástrojů v souboru SKILL.md.

Architektura „hub-and-spoke"

Gateway – Control Panel (WebSocket, WhatsApp, Telegram, Slack, WebUI)
Brain – orchestrace LLM přes ReAct smyčku
Memory – persistentní kontext v Markdown
Skills – zásuvné moduly pro akce
Heartbeat – plánovač úkolů

Jak vybrat správný framework

Maximální kontrola a auditovatelnost workflow → LangGraph.

Rychlé nasazení týmu agentů s minimem kódu → CrewAI.

Produkční spolehlivost a validace výstupů → Pydantic AI.

Microsoft/Azure prostředí → Microsoft Agent Framework (nástupce AutoGen + Semantic Kernel); Google Cloud → Google ADK.

RAG a práce s dokumenty → LlamaIndex.

Výkon a multimodalita → Agno (dříve Phidata).
